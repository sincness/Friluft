# kommandoer
### Start både clienten & serveren
> npm run dev

### Start server (Node)
> npm run server

### Start client (React)
> npm run client

### Server 
* Serveren kører på http://localhost:5000 
* og clienten kører på http://localhost:3000 

Åben http://localhost:3000 for at se projektet i browseren.
